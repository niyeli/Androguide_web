 使用技術

 開發應用：機器學習（Find Datasets AndTraining Model ）/ Feature Extraction / python 3

 開發工具: Androguard

 網站：PHP MYSQL HTML Apache

 版本控管：Github
 系統簡介  一個擁有檢測apk是否藏有病毒的偵測工具，透過使用者上傳apk檔提供我們做檢測，避免讓手機中毒的風險
 平台價值  透過免費提供一個專業的平台檢測APK是否藏有病毒，也打造一個資安知識的整理懶人包，更提供由初學者到高階不同程度的資安資源的分類，不論是任何想透過此網站接觸資安訊息的使用者都可以輕鬆取得相對的資訊
 功能說明  只要使用者將尚未安裝的手機軟體APK檔上傳到我們的網頁，只要不到一分鐘的時間，網站就能判定此軟體是否可以放心安裝，避免蘊藏病毒感染的風險
